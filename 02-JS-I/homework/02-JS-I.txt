Variables 
Una variable es una forma de almacenar algo que podemos configurar y reestablecer para utilizar luego.
Para crearla utilizamos la palabra clave var

Strings
Son bloques de texto siempre definidas entre comillas

Funciones (argumentos, `return`)
Las funciones son tipos particulares de objetos llamados invocables. 
Sirven para calcular las variables, cambiarlas, hacer algo con ellas

Argumentos 
Son los datos sobre los que va a trabajar la funcion y se pueden agregar multiples utulizando una coma.

Return
Lo utilizamos cuando queremos devolver algo, si luego de una operacion matematica utilizamos "return" devolvera el resultado de la operacion y no se podra modificar.

Declaraciones 
Es una afirmacion o valor que le damos a las variables

if es el operador que determina condicion y esta se resuelve con los datos de la declaracion

Valores Booleanos
Se utilizan en JavaScript significan verdadero o falso (true o false)
